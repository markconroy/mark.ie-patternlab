    @mixin icon($icon, $pseudo: before, $text-replace: false, $size: inherit, $color: inherit, $block: false) { ... }
    View source

    Description

    Main Icon mixin. See the Icons in Pattern Lab.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $icon

    Machine name of icon (filename).

    		String— none
    $pseudo

    before | after The pseudo element to place the icon in.

    		Stringbefore
    $text-replace— noneBoolfalse
    $size— noneStringinherit
    $color— noneStringinherit
    $block

    If display: block should be applied.

    		Boolfalse

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

  @include icon('close');
}
